
SpartanRPC README.txt
=====================

This folder contains a staged SpartanRPC sample. This sample generates a program that...

In addition this sample demonstrates how Scalaness generated code can interact with existing
(non-staged) libraries.

The nbproject folder is a NetBeans free form project. It uses the build.xml file in the
Scalaness root folder for building the Blink sample with the latest Scalaness plugin.

In order for NetBeans to properly locate scala-compiler.jar and scala-library.jar you need to
copy the file scala.properties-sample to scala.properties and then edit the copy as appropriate.
