Scalaness

Scalaness is a modified Scala compiler that supports a form of staged programming for embedded systems developers using nesC. The Scalaness compiler allows the user to specialize and compose nesC modules. One unique aspect of the Scalaness system is that if the Scalaness program type checks then every possible generated nesC program will also type check.

The Scalaness source code is currently on GitHub
