Scalaness/nesT

Code Examples and Compiler Download

Overview

This web page provides supplementary materials for the paper entitled Scalaness/nesT: Type Specialized Staged Programming for Sensor Networks, submitted to LCTES 2013. We have included two working program code examples, two code examples that contain type errors caught by our type analysis, and a link to GitHub the latest Scalaness/nesT compiler is located. See the Scalaness README for more information about the samples currently in the GitHub repository and how to run them.

Compiler Source Code

The source code of the Scalaness/nesT compiler, examples, and associated documentation is available on GitHub. The valid code examples above can be successfully compiled with this compiler, and it will detect type errors in the invalid code examples. Please see the main README file for instructions for building the compiler.
